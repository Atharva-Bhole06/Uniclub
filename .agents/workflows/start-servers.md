---

// turbo-all

1. Start the React frontend server
Use the `run_command` tool to run `npm run dev` in the `d:\Atharva\JavaProject\Uniclub\frontend` directory.

2. Start the Spring Boot backend server
Use the `run_command` tool to run `.\mvnw spring-boot:run` in the `d:\Atharva\JavaProject\Uniclub\backend` directory.

3. Complete
Inform the user that the servers have started and that the frontend is accessible at [http://localhost:5173/](http://localhost:5173/) and the backend at http://localhost:8080/ .
